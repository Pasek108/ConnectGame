Connect Game
Artur Pas
Zasady Rules
Celem gry jest zbudowanie mostów miedzy wyspami. Zadady gry:
  • Wyspy można łączyć tylko pionowo i poziomo do najbliższej wyspy
  • Każda strona wyspy może mieć brak, jedno lub dwa połączenia
  • Liczba mostów połączonych do wyspy musi byc równa liczbie na wyspie
  • Mosty nie mogą się krzyżować

The goal is to connect all of the islands by drawing a series of bridges between the islands.

The bridges must follow certain criteria:
  • They must begin and end at distinct islands, travelling a straight line in between.
  • They must not cross any other bridges or islands.
  • They may only run orthogonally (i.e. they may not run diagonally).
  • At most two bridges connect a pair of islands.
  • The number of bridges connected to each island must match the number on that island.
  • The bridges must connect the islands into a single connected group.
Celem gry jest podłączyć wszystkie punkty końcowe do źródła obracając pojedyńcze płytki. Zadady gry:
  • Każdy punkt końcowy musi być połączony ze źródłem
  • Każda ścieżka musi kończyć się punktem końcowym
 The goal of the game is to connect all endpoints to the source by rotating tiles. Rules:
  • Every endpoint must be connected to the source
  • Every path must end by endpoint
Celem gry jest podłączyć wszystkie punkty końcowe do źródła manipulując siatką za pomocą przesuwania wierszy i kolumn. Zadady gry:
  • Każdy punkt końcowy musi być połączony ze źródłem
  • Każda ścieżka musi kończyć się punktem końcowym
 The goal of the game is to connect all endpoints to the source by sliding rows and columns. Rules:
  • Every endpoint must be connected to the source
  • Every path must end by endpoint
Celem gry jest odtworzenie poprawnej sieci połączeń między blokami. Sieć jest poprawna gdy wszystkie połączenia bloku są połączone z taką samą liczbą połączeń bloków sąsiednich oraz możliwe jest przejście z dowolnego bloku do każdego innego.

W tym trybie występują 4 bloki:

 The goal of the game is to recreate correct net of connections beetwen blocks. Net is correct when all connections of the block are connected with the same number of connections of his adjacent blocks and it is possible to travel from any block to other.

This mode has 4 type of blocks:
  • Blok bez ruchu, nic nie można z nim zrobić Block with no move, you can do nothing with it
  • Blok z obrotem, można go obracać za pomocą kliknięcia Block with rotation, you can rotate it by clicking
  • Blok z ruchem pionowym i poziomym, można go przesuwać w dowolnym kierunku lecz nie można go obracać Block with vertical and horizontal movement, you can drag it around but you cant rotate
  • Blok z obrotem oraz ruchem pionowym i poziomym, można go przesuwać w dowolnym kierunku i obracać Block with rotation, vertical and horizontal movement, you can drag it around and rotate
