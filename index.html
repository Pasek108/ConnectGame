Zasady Rules
Celem gry jest odtworzenie poprawnej sieci połączeń między blokami. Sieć jest poprawna gdy wszystkie połączenia bloku są połączone z taką samą liczbą połączeń bloków sąsiednich oraz możliwe jest przejście z dowolnego bloku do każdego innego.

W tym trybie występują 4 bloki:

 The goal of the game is to recreate correct net of connections beetwen blocks. Net is correct when all connections of the block are connected with the same number of connections of his adjacent blocks and it is possible to travel from any block to other.

This mode has 4 type of blocks:
  • Blok bez ruchu, nic nie można z nim zrobić Block with no move, you can do nothing with it
  • Blok z obrotem, można go obracać za pomocą kliknięcia Block with rotation, you can rotate it by clicking
  • Blok z ruchem pionowym i poziomym, można go przesuwać w dowolnym kierunku lecz nie można go obracać Block with vertical and horizontal movement, you can drag it around but you cant rotate
  • Blok z obrotem oraz ruchem pionowym i poziomym, można go przesuwać w dowolnym kierunku i obracać Block with rotation, vertical and horizontal movement, you can drag it around and rotate
